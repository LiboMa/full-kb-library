## 🧠 核心内容

### 一、什么是趋势 (Trend)

| 趋势类型 | 定义 | 特征 |
|----------|------|------|
| **上升趋势 (Uptrend)** | 价格形成连续的更高的高点 (Higher Highs) 和更高的低点 (Higher Lows) | 买方力量 > 卖方力量 |
| **下降趋势 (Downtrend)** | 价格形成连续的更低的高点 (Lower Highs) 和更低的低点 (Lower Lows) | 卖方力量 > 买方力量 |
| **横盘/震荡 (Sideways/Range)** | 价格在支撑和阻力之间来回震荡 | 买卖力量均衡 |

### 二、趋势的三个时间维度

```
长期趋势 (Primary Trend)    → 周线/月线 → 数月至数年
中期趋势 (Secondary Trend)  → 日线      → 数周至数月
短期趋势 (Minor Trend)      → 小时线    → 数天至数周
```

**Adam Khoo 核心观点**：不同时间框架的趋势可以同时存在且方向相反。做交易时，必须明确你关注的是哪个时间框架的趋势。

### 三、如何画趋势线 (Trendline)

#### 上升趋势线：
1. 找到至少 **2-3 个明显的低点 (Swing Lows)**
2. 用直线连接这些低点
3. 价格在趋势线上方运行 = 上升趋势持续
4. 价格跌破趋势线 = 趋势可能反转的信号

#### 下降趋势线：
1. 找到至少 **2-3 个明显的高点 (Swing Highs)**
2. 用直线连接这些高点
3. 价格在趋势线下方运行 = 下降趋势持续
4. 价格突破趋势线 = 趋势可能反转的信号

### 四、趋势交易的核心原则

> **"不要逆势交易。" — Adam Khoo**

| 原则 | 说明 |
|------|------|
| **顺势交易 (Trade with the trend)** | 上升趋势做多，下降趋势做空 |
| **回调入场 (Buy the dip in uptrend)** | 在趋势方向的回调中寻找入场点 |
| **不在横盘中追涨杀跌** | 横盘时等待突破方向确认 |
| **趋势末期减仓** | Higher Highs 变弱、成交量背离时警惕 |

### 五、趋势强度判断

**趋势越强的特征：**
- 价格远离趋势线（强势拉升）
- 回调幅度小（浅回调 = 强趋势）
- 成交量在趋势方向放大
- 移动平均线呈多头/空头排列

**趋势减弱的信号：**
- 回调幅度加深
- 新高/新低缺乏力度
- 成交量萎缩
- 趋势线斜率变平

### 六、趋势反转 vs 趋势回调

| | 回调 (Pullback) | 反转 (Reversal) |
|--|----------------|----------------|
| **幅度** | 小 (通常 38.2%-50% 回撤) | 大 (超过前一波段 61.8%+) |
| **时间** | 短暂 | 持续 |
| **成交量** | 缩量 | 放量 |
| **结构** | 不破前低/前高 | 打破 HH/HL 或 LH/LL 结构 |

## 🎯 关键 Takeaways

| # | 要点 |
|---|------|
| 1 | **趋势是技术分析的基础** — 先判断趋势，再决定方向 |
| 2 | **Higher Highs + Higher Lows = 上升趋势** — 这是最基本的判断标准 |
| 3 | **顺势交易胜率最高** — 不要试图"抄底"或"摸顶" |
| 4 | **趋势线需要至少 3 个触碰点** — 2 点画线，3 点确认 |
| 5 | **多时间框架验证** — 短期趋势应与长期趋势方向一致 |

## 💡 实战应用

```
交易流程：
1. 判断长期趋势方向（周线）
2. 在日线上等待回调至趋势线/支撑区域
3. 确认反弹信号（K线形态 + 指标）
4. 顺势入场
5. 止损设在趋势结构破坏点下方
```
