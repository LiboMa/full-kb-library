## 📋 课程定位

本课为系列第五课，专门讲解**日内交易 (Day Trading)** 的核心策略——如何在盘中获取利润并在收盘前平仓。

---

## 🧠 核心内容

### 一、日内交易 vs 波段交易

| 特征 | 日内交易 (Day Trading) | 波段交易 (Swing Trading) |
|------|----------------------|------------------------|
| **持仓时间** | 分钟至小时，当日平仓 | 数天至数周 |
| **时间框架** | 1分钟、5分钟、15分钟 K线 | 日线、周线 |
| **交易频率** | 每天多笔 | 每周数笔 |
| **利润目标** | 小额但高频 | 单笔较大 |
| **资金要求** | 美股 PDT 规则要求 $25,000+ | 无最低要求 |
| **心理压力** | 高（需快速决策） | 中等 |
| **适合人群** | 全职交易员 | 有本职工作者 |

### 二、日内交易的核心要素

#### 1. 选择合适的股票
```
日内交易选股标准：
✅ 高流动性 (日均成交量 > 100万股)
✅ 适度波动性 (日均波幅 2-5%)
✅ 有催化剂 (财报、新闻、板块异动)
✅ 干净的图表形态 (清晰的支撑/阻力)
❌ 避免: 低流动性小盘股、价差过大的股票
```

#### 2. 时间框架选择
- **5分钟 K线**: Adam Khoo 最常用的日内时间框架
- **1分钟 K线**: 用于精确入场/出场
- **15分钟 K线**: 判断盘中趋势方向
- **日线**: 确定当天的大方向和关键支撑/阻力

### 三、Adam Khoo 的日内交易策略

#### 策略一：动量突破 (Momentum Breakout)

```
条件：
1. 股票在盘前有重大消息/催化剂
2. 开盘后形成窄幅盘整 (Consolidation)
3. 成交量放大
4. 突破盘整区域上沿

入场: 突破确认（收盘突破）
止损: 盘整区域下沿
止盈: 1:2 或 1:3 风险收益比
```

#### 策略二：回调入场 (Pullback Entry)

```
条件：
1. 确认盘中趋势方向（5分钟 Higher Highs / Higher Lows）
2. 等待价格回调至 20 EMA 或短期支撑
3. 出现反弹确认 K线（锤子线、吞没形态）

入场: 确认 K线收盘后
止损: 回调低点下方
止盈: 前高或 1:2+ 盈亏比
```

#### 策略三：开盘区间突破 (Opening Range Breakout - ORB)

```
规则：
1. 观察开盘后 15-30 分钟的价格区间
2. 标记该区间的高点和低点
3. 突破高点 → 做多 | 跌破低点 → 做空

入场: 突破 + 成交量确认
止损: 区间中间位置或对面边界
止盈: 区间宽度的 1.5-2 倍
```

### 四、日内交易的风险管理

| 规则 | 参数 |
|------|------|
| 单笔风险 | ≤ 账户的 1% |
| 每日最大亏损 | ≤ 账户的 3% |
| 连续亏损后 | 停止交易，复盘原因 |
| 盈亏比 | 最低 1:2 |
| 每日交易次数 | 上限 3-5 笔（质量 > 数量） |

> **Adam Khoo 核心原则**: "如果早盘连续亏损 2 笔，当天就停止交易。市场不适合你当前的状态。"

### 五、盘前准备 (Pre-Market Routine)

```
开盘前 30 分钟：
1. 检查隔夜新闻和期货走势（判断大盘方向）
2. 筛选有催化剂的异动股（盘前涨/跌幅 > 3%）
3. 画出当天关键支撑/阻力位
4. 设定交易计划：哪些股票、什么条件入场、止损止盈在哪
5. 心理准备：接受今天可能不交易
```

### 六、常见陷阱

| 陷阱 | 解决方案 |
|------|----------|
| 过度交易 (Overtrading) | 限制每日交易次数 |
| 报复性交易 | 设定每日亏损上限后强制停止 |
| FOMO (错失恐惧) | 没有满足所有条件就不入场 |
| 移动止损太紧 | 给交易"呼吸空间" |
| 在震荡市中追突破 | 确认趋势日才做突破 |

## 🎯 关键 Takeaways

| # | 要点 |
|---|------|
| 1 | **日内交易需要全职投入** — 不适合有本职工作的人 |
| 2 | **流动性是第一要求** — 没有足够成交量的股票不要碰 |
| 3 | **纪律比技术更重要** — 严格执行每日亏损上限 |
| 4 | **少即是多** — 每天 2-3 笔高质量交易 > 10笔随意交易 |
| 5 | **盘前准备决定当天成败** — 开盘后不应该才开始找机会 |

---

## ⚠️ 重要提醒

> Adam Khoo 多次强调：**日内交易不适合大多数人。** 如果你有全职工作，波段交易 (Swing Trading) 是更好的选择。日内交易需要：
> - 至少 $25,000 资金（美股 PDT 规则）
> - 全天盯盘的时间
> - 极强的心理素质和纪律性
> - 至少 6 个月的模拟盘练习
