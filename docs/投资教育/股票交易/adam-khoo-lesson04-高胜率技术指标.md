## 🧠 核心内容

### 一、技术指标的本质

> **指标不是魔法工具，而是价格数据的数学变换。** 它们帮助你更清晰地"看见"趋势、动量和潜在反转，但不能预测未来。

**Adam Khoo 的指标哲学：**
- 指标是**确认工具 (Confirmation Tool)**，不是决策工具
- 价格行为 (Price Action) 永远优先于指标
- 少即是多 — 2-3 个互补指标足够

### 二、移动平均线 (Moving Averages)

| 类型 | 公式特点 | 用途 |
|------|----------|------|
| **SMA (Simple MA)** | 等权重平均 | 判断长期趋势方向 |
| **EMA (Exponential MA)** | 近期权重更高 | 更灵敏，适合短期交易 |

#### Adam Khoo 常用设置：
```
短期: 20 EMA → 短期趋势方向
中期: 50 EMA → 中期趋势方向  
长期: 150 SMA / 200 SMA → 长期趋势方向
```

#### 移动平均线交易信号：
- **金叉 (Golden Cross)**: 短期 MA 上穿长期 MA → 看涨
- **死叉 (Death Cross)**: 短期 MA 下穿长期 MA → 看跌
- **价格在 MA 上方**: 趋势看涨，MA 作为动态支撑
- **价格在 MA 下方**: 趋势看跌，MA 作为动态阻力

### 三、MACD (Moving Average Convergence Divergence)

#### 组成：
```
MACD Line = 12 EMA - 26 EMA
Signal Line = MACD Line 的 9 EMA
Histogram = MACD Line - Signal Line
```

#### 交易信号：
| 信号 | 含义 | 操作 |
|------|------|------|
| MACD 上穿 Signal Line | 动量转多 | 买入信号 |
| MACD 下穿 Signal Line | 动量转空 | 卖出信号 |
| Histogram 由负转正 | 多头动量增强 | 确认买入 |
| **MACD 背离 (Divergence)** | 价格与动量不同步 | ⚠️ 反转预警 |

#### 背离的力量：
```
看涨背离: 价格创新低，但 MACD 没创新低 → 下跌动量衰竭
看跌背离: 价格创新高，但 MACD 没创新高 → 上涨动量衰竭
```

### 四、RSI (Relative Strength Index)

#### 基本参数：
- 周期: 14（默认）
- 范围: 0-100
- 超买: > 70 | 超卖: < 30

#### Adam Khoo 的 RSI 用法：

| 场景 | RSI 位置 | 操作 |
|------|----------|------|
| 上升趋势中 RSI 回到 40-50 | 回调至"价值区域" | 顺势买入 |
| 下降趋势中 RSI 反弹到 50-60 | 反弹至"价值区域" | 顺势做空 |
| RSI 背离 | 与价格方向不同步 | 警惕反转 |

> **注意**: Adam Khoo 强调不要单纯因为 RSI>70 就做空或 RSI<30 就做多。在强趋势中，RSI 可以长时间保持在超买/超卖区域。

### 五、随机指标 (Stochastic Oscillator)

#### 基本参数：
```
%K = 14 期随机值
%D = %K 的 3 期移动平均
范围: 0-100
超买: > 80 | 超卖: < 20
```

#### 交易方式：
- 在**超卖区域 %K 上穿 %D** → 买入信号（需结合趋势方向）
- 在**超买区域 %K 下穿 %D** → 卖出信号
- 与 MACD 配合使用效果更佳

### 六、布林带 (Bollinger Bands)

#### 组成：
```
中轨 = 20 SMA
上轨 = 中轨 + 2 × 标准差
下轨 = 中轨 - 2 × 标准差
```

#### 应用场景：
| 形态 | 含义 | 策略 |
|------|------|------|
| 价格触及下轨 + 趋势向上 | 超卖回调 | 买入机会 |
| 价格触及上轨 + 趋势向下 | 超买反弹 | 做空机会 |
| 带宽收窄 (Squeeze) | 波动率降低，即将突破 | 准备跟进突破方向 |
| 带宽扩张 | 波动率增大 | 趋势加速中 |

## 🎯 关键 Takeaways

| # | 要点 |
|---|------|
| 1 | **指标是确认工具，不是预测工具** — 先看价格行为，再用指标确认 |
| 2 | **不要过度堆叠指标** — 2-3 个互补指标即可（趋势型 + 动量型） |
| 3 | **背离是最强的反转信号** — MACD/RSI 与价格的背离值得高度关注 |
| 4 | **指标必须结合趋势使用** — 超买不等于必跌，超卖不等于必涨 |
| 5 | **移动平均线排列判断趋势方向** — 多头排列做多，空头排列做空 |

## 📐 Adam Khoo 推荐指标组合

```
趋势判断:  20 EMA + 50 EMA + 150 SMA
动量确认:  MACD (12, 26, 9)
超买超卖:  Stochastic (14, 3) 或 RSI (14)
波动率:    Bollinger Bands (20, 2)

入场条件 = 趋势方向 ✅ + 价格行为确认 ✅ + 指标确认 ✅
```
