## 🧠 核心内容（基于作者原话）

### 一、支撑位与阻力位的本质

- **支撑位和阻力位 = 供需区域 (Supply & Demand Zones)**
- 价格在这些水平倾向于停止并反转，原因是供需失衡
- Adam Khoo 强调这不是精确的"线"，而是**区域 (Zones)**

### 二、如何识别支撑位

1. **寻找价格汇聚点** — 多次在同一价位附近止跌回升
2. **画线连接至少 3 个触碰点** — 至少 3 次验证才可靠
3. **示例**: 价格在 $66 附近多次止跌 → $66 为支撑区域
4. **交易策略**: 在支撑位附近买入（需确认反弹信号）

### 三、如何识别阻力位

1. **在下降趋势中寻找趋势线** — 连接至少 3 个高点
2. **价格反弹至阻力位时考虑做空** — 当价格再次触及同一水平
3. **交易策略**: 在阻力位做空或平仓多头

### 四、支撑阻力互换原则（角色转换）

> **关键概念**: 一旦阻力位被突破，它会变成新的支撑位；反之亦然。

- 突破阻力 → 该水平成为新支撑 → 回踩确认后可买入
- 跌破支撑 → 该水平成为新阻力 → 反弹至此可做空

### 五、确认与过滤

- **结合移动平均线和其他技术指标确认** — 不要单独依赖支撑阻力
- **避免假突破 (False Breakouts)** — 等待收盘确认
- **避免在盘整区间 (Consolidation) 中交易** — 等待明确方向

### 六、核心方法论

> "识别可重复的价格形态 (Repeatable Patterns)，做出有据可依的决策，最大化利润。"

## 📐 实战应用框架

```
1. 画出关键水平（至少 3 点验证）
2. 判断趋势方向（上升/下降/横盘）
3. 等待价格接近关键水平
4. 确认信号（K线形态 + 指标）
5. 入场 + 设止损在水平另一侧
```
