## 三个常见问题

**Q1：Hermes Agent 和 Hermes 3 LLM 是一回事吗？**
不是。Hermes 3 是模型（发动机），Hermes Agent 是框架（车）。本目录只讲车。

**Q2：要先装好才能读吗？**
Part 1 建议边读边装（10 分钟）。Part 2 配合实操效果最好。Part 3 纯原理，不装也能读。

**Q3：读完能做什么？**
- 读完 Part 1：把 Hermes 当 ChatGPT + 文件读写 + 代码执行来用
- 读完 Part 2：跑定时任务、接 Telegram/飞书、写 Skill 沉淀流程、派子 agent 并行
- 读完 Part 3：改源码、加 provider、把 Hermes 嵌进自己项目
- 读完附录 A：对着 12 个场景挑最像自己需求的照搬
